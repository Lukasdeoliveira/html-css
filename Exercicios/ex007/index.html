Principais formatações

Negrito / Destaque

Para por a mensagem em Negrito você utiliza a tag <Strong>

Italico / Ênfase

Para por a mensagem em Italico Você utiliza a tag <em>

Quero marcar uma parte do texto IMPORTANTE do texto

Agora se você quiser marcar a palavra IMPORTANTE em outra cor ultilize o parametro style (css) dentro da tag mark

Agora se eu utilizar a tag style no meu ' head ', com o mark e o background eu mudo a cor da marcação IMPORTANTE de todas as tag mark

Texto pequeno

Para deixar uma parte do texto pequeno você deve usar a tag <small>. Isto serve para por exemplo como letras pequenas num contrato, logo então é semantico para o HTML.

Texto deletado

Para excluir uma parte do texto utilizamos a tag <del> para ler a parte deletada e não a considerar

Texto inserido (sublinhado)

Para por um texto inserido (semantico) (sublinhado) utilizamos a tag <ins>

Texto sobrescrito

Para inserir coisas tipo x52+3. Utilizamos a tag <sup>

Texto subscrito

Para inserir coisas tipo H2O. Utilizamos a tag <sub>
